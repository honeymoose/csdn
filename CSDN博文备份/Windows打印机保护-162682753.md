<p>Windows 又开始盯上了打印机，为打印机提供了一个保护打印机模式。</p> <br><p></p> <br><p></p> <br><p class="img-center"><a href="https://cdn.isharkfly.com/com-isharkfly-www/discourse-uploads/original/3X/4/c/4c28688c08016953eb47f9ae28c626fcf3c8d7db.jpeg" rel="nofollow"><img alt="2026-07-07_18-16-40" height="500" src="https://i-blog.csdnimg.cn/img_convert/666fddffb40ffad4092056014b727e8b.jpeg" width="621" /></a></p> <br><p></p> <br><p>Windows Protected Print（Windows 受保护打印）是微软从 Windows 11 24H2 开始推出的一种新的打印架构（Windows Protected Print Mode，简称 <strong>WPP</strong>），目的是提高打印的安全性、稳定性和兼容性。</p> <br><h3>推出的原因</h3> <br><p>传统 Windows 打印依赖大量第三方打印驱动（Printer Driver），这些驱动：</p> <br><ul><li>运行在系统内核或高权限环境</li><li>容易导致蓝屏（BSOD）</li><li>存在安全漏洞</li><li>每个厂商都需要维护自己的驱动</li></ul> <br><p>WPP 希望让 Windows 不再依赖这些专有驱动。</p> <br><p>Windows 推出这个保护打印机模式的原因还是因为安全性的问题，例如 Print Spooler 提权漏洞等等，很多漏洞最近都是从打印机这边开始的。</p> <br><p>Windows 希望通过这个保护打印机模式，来对打印机机的底层进行一个隔离。</p> <br><p>Windows 也希望通过自己把打印机的驱动全部管起来，那么这样打印机的硬件生产厂商就只负责打印机，至于驱动方面就全部交给 Windows。</p> <br><p>对于最终用户来说没有什么大的影响，老的打印机驱动还是可以用的，只是针对打印机保护会在界面中显示出一个小的图标。</p> <br><p></p> <br><p><a href="https://www.isharkfly.com/t/windows/2323" title="Windows 保护打印机 - DevOps - iSharkFly" rel="nofollow">Windows 保护打印机 - DevOps - iSharkFly</a></p> <br><p></p>
